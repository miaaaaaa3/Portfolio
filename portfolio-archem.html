Architecture Emulator: Details

One of the first real computer architecture projects I attempted in the Fall 2022 was this. I wrote an instruction set emulator for a small subset of the A64 instruction set of the AArch64 execution state of the Armv9 architecture. Basically facilitating instructions for assembly code. Very cool.
